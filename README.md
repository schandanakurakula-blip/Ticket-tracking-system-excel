# Ticket Tracking System (Excel-Based)

## Overview
This project is an Excel-based ticket tracking system designed to manage and monitor client issues efficiently. It simulates real-world operations support workflows.

## Features
- Track client issues with Ticket ID, Priority, and Status
- Dropdown-based status and priority management
- Dashboard with ticket status visualization
- 25+ sample records for realistic data simulation

## Tools Used
- Microsoft Excel
- Pivot Tables
- Data Validation
- Charts

## Purpose
This project demonstrates operations support, workflow management, and data analysis skills relevant to business and IT roles.
